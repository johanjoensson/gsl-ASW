Detailed Description

A class for representing atoms in the cell.
Contains:
Z - Nuclear charge
AS - Atomic sphere radius
MT - Muffin tin radius
pos - position, in cartesian coordinates
mesh - logarithmic mesh to use in intraatomic calculations
