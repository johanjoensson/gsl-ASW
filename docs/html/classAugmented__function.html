Detailed Description

Superclass used to describe the general behaviour of an augmented function.
Contains:
n - Principal quantum number of the augmented function.
l - Combined orbital and angular momentum of the augmented function, l = {l,m}.
kappa - Energy of waves outside augmentation region, used for matching at sphere boundary.
radius - Radius of augmentation sphere.
center - Position of the center of the augmentation sphere.
mesh - Logarithmic mesh of the augmentation sphere.
val - values of the augmented function at the mesh points.
