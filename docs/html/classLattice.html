Detailed Description

A class used to represent lattices, e.g. Bravais lattices useful in describing crystals.
Contains:
lat - GSL::Matrix constructed from the three lattice vectors (row major).
r_lat - GSL::Matrix constructed from the three reciprocal lattice vectors (row major).
scale - Scale factor to multiply each lattice vector with.
volume - Volume of the unit cell.
bz_volume - Volume of the first Brillouin zone.
