Detailed Description

Class for representing an augmented spherical wave.
Contains:
center - Atomic site the wave is centered on, on-center augmentation sphere.
off_centers - Atomic sites where the wave is augmented in off-center spheres.
__kappa_ - Energy of waves in interstitial region.
n - principal quantum number of the wave.
l - Combined orbital and magnetic quantum number.
s - Spin direction.
__core_state_ - Is this wave describing a core state or not? Core states have no augmentation in off-center spheres.
H - On center, augmented Hankel function.
J - Off-center, augmented Bessel functions (unsorted).
