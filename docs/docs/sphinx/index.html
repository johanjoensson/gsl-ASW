bool operator<(const lm &a) const¶

Ordering according to the Aufbauprinzip i.e., first according to n + l, if n + l equal, order according to n. If n are equal order according to m (ordering according to m is not part of the Aufbauprinzip). E.g. [1s, 2s, 2p, 3s, 3p, 4s, 3d, 4p, …]
